About Website

Student Resource Management System is a one stop destination which allows you to search, preview and download various textbooks, PDF's previous papers and other competitive exam resources that helps students for their preparation.While large in scope sometimes searching for a single book in an entire library might be hectic task and this application will help in saving time and effort of student.Rather exploring the entire library, the resources that a student needs can be downloaded in just one mouse click. The ultimate goal is to build a platform that provides universal access to resources and knowledge to every student..
